# Marina Silva Diniz - Portfolio

A professional portfolio website built to present my experience as a mobile and frontend developer. The project highlights my work with React Native, TypeScript, Swift, and web technologies, while also giving visitors a clear way to explore projects and get in touch.

## About the project

This portfolio was designed as a clean, modern, and responsive single-page experience. It includes an introduction section, an about section, featured projects, and a contact form connected to Formspree.

## Features

- Responsive layout for desktop and mobile
- Hero section with personal introduction and social links
- About section with background, skills, and highlights
- Projects section with selected work and external links
- Contact form with real submission via Formspree
- GitHub Pages deployment support

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React
- Formspree
- GitHub Pages

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Deploy to GitHub Pages:

```bash
npm run deploy
```

## Project structure

- `src/app/App.tsx` - Main portfolio page and sections
- `src/main.tsx` - Application entry point
- `src/styles/` - Global styles and theme files
- `assets/` - Static files such as the profile image and favicon

## Deployment

The project is configured for GitHub Pages through the `homepage` field in `package.json` and the `deploy` script.

This project was based on a Figma layout and adapted into a working portfolio website.
